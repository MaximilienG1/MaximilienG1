## Selected Project Contributions

### [Opti4ML-TidyTeam](https://github.com/Reza-rgb/Opti4ML-TidyTeam) 
*Spring 2025*
Contributed to a machine learning project investigating how data ordering and sampling strategies affect training performance, with experiments on synthetic data, MNIST, and CIFAR-10. The project compared curriculum learning, reverse curriculum learning, hard example mining, and stratified Monte Carlo sampling, including robustness analyses under Gaussian and impulse noise. 


### [ML4Science-CS433](https://github.com/MetthewB/ML4Science-CS433)
*Autumn 2024*  
Contributed to an image denoising and evaluation project comparing Gaussian and median filtering methods on multi-channel image data. The project focused on processing `.npy` microscopy-style images and assessing filter performance with PSNR and SSIM metrics.


### [LLMFoodEmbeddingScores](https://github.com/digitalepidemiologylab/LLMFoodEmbeddingScores)
*Spring 2024*
Contributed to a project exploring food representation and scoring methods using large language models and embedding-based approaches.
